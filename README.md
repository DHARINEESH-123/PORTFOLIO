## 🛠️ How to Deploy

### GitHub Pages (Recommended)
1. Initialize Git locally and push this project directory to your personal GitHub repository.
2. Go to the repository **Settings** -> **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch** and choose your deploy branch and `/root` directory.
4. Click **Save**; your portfolio will be live in minutes.
